This is a Mixed Reality (MR) minigolf experience designed for the **Meta Quest** platform.  
The game blends the player's real environment with procedurally generated golf courses, allowing players to interact with virtual clubs and balls inside their physical space.

This project is developed in **Unity** as part of an Introduction to AR course project.
